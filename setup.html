How to Install Python

Step 1: Download Python

Visit the official Python website to download Python.

Step 2: Run the Installer

Locate the downloaded file and double-click it to run the installer.

Step 3: Setup Options

Check the "Add Python to PATH" option and click "Install Now".

Step 4: Installation Progress

Click the "Install" button to begin the installation.

Step 5: Verify the Installation

Open Command Prompt and type python --version to verify that Python is installed correctly.
